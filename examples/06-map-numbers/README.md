# Map Numbers Example

Demonstrates the **Array Map** loop node with numeric transformation.

## What it does

A `doubleArray()` function that takes an array of numbers and returns a new array where each element is multiplied by 2.

```javascript
doubleArray([1, 2, 3, 4, 5]) // returns [2, 4, 6, 8, 10]
```

## Visual Flow

- **Map Node**: Iterates over the input array
- **Multiply Operator**: Multiplies each element by 2 (using a wired literal)
- **Return Node**: Captures the mapped result

This example shows the fully wired approach — no callback code editor, all logic is visual.
