# Using NodePreview in Documentation

The `NodePreview` component renders a live, embedded preview of any node type. Instead of static screenshots, docs show the real, current node UI.

## Basic Usage

To embed a node preview in your documentation, use the `NodePreview` component with a `type` prop:

```mdx
<NodePreview type="express.route" />
```

This renders the node with default/empty configuration.

## With Configuration

Pass a `data` object to show the node with specific configuration:

```mdx
<NodePreview 
  type="express.route" 
  data={{ path: "/api/users", method: "GET" }}
/>
```

## Examples

### Server Nodes

<NodePreview type="express.init" />

<NodePreview type="express.listen" data={{ port: 3000 }} />

### Routing

<NodePreview 
  type="express.route" 
  data={{ path: "/users/:id", method: "POST" }}
/>

### Control Flow

<NodePreview 
  type="controlFlow.branch" 
  data={{ condition: "score > 80" }}
/>

<NodePreview 
  type="controlFlow.sequence"
  data={{ pins: ["then-0", "then-1", "then-2"] }}
/>

### Operators

<NodePreview type="operators.add" />

<NodePreview 
  type="operators.equal" 
  data={{ strict: false }}
/>

### Variables

<NodePreview 
  type="variable.get" 
  data={{ variableId: "var_example" }}
/>

<NodePreview 
  type="variable.set" 
  data={{ variableId: "var_example", variableType: "string" }}
/>

### Logic

<NodePreview 
  type="logic.function" 
  data={{ name: "calculateTotal", params: ["items", "tax"] }}
/>

<NodePreview 
  type="logic.require" 
  data={{ path: "lodash", isLocal: false }}
/>

## Customization

You can control the iframe height and width:

```mdx
<NodePreview 
  type="express.route"
  data={{ path: "/search", method: "GET" }}
  height={600}
  width="100%"
/>
```

The default height is 500px and width is 100%.

## Benefits

- **Always current**: Updates automatically when node styling/UI changes
- **Real component**: Shows the exact visual behavior from the editor UI
- **No maintenance**: No need to regenerate screenshots manually
- **Discoverable**: Readers can see exactly how each node looks
