# Configuration

How visual-node's editor is configured: environment variables, project directory structure,
and execution mode settings.

- [Environment Variables](/configuration/environment-variables) — `PORT` and
  `FLOWSERVER_PROJECT_DIR`, and the precedence between a CLI argument and the env var.
- [Project Directory](/configuration/project-directory) — what visual-node expects to
  find (and will create) inside your project directory, and how multi-file projects work.
- [Project Settings](/configuration/project-settings) — configuring Server vs. Script
  execution mode via the Settings modal in the editor toolbar.
