# Boolean

A **Boolean** variable holds `true` or `false`. Instead of a free-text field, the
Variables panel shows a small true/false selector for its default value, so there's no
way to type anything else in by mistake.

## Compiles to

```js
let isActive = true;
```

## Get Variable / Set Variable

A **Get isActive** node reads it back; a **Set isActive** node assigns a new value and
continues:

## Use case

- **Flags and state**: A feature flag, an "is authenticated" state, or any other on/off
  switch you want to check and flip from the canvas.
