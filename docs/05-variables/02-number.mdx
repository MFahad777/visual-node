# Number

A **Number** variable holds a numeric value. The default-value text can be any integer
or decimal — it's validated as a real, finite number, and rejected if it isn't (no
quotes, no stray characters, just the digits).

## Compiles to

```js
let counter = 0;
```

Typing `0` into the default-value field emits a bare numeric literal — no quotes, no
wrapping.

## Get Variable / Set Variable

A **Get counter** node reads it back; a **Set counter** node assigns a new value and
continues:

## Use case

- **Counters and limits**: A request counter, a retry limit, or a page-size default —
  anything you want to read, increment, or reset from the canvas instead of hardcoding
  it inline.
