# visual-node documentation

This is the [Docusaurus](https://docusaurus.io/) site for visual-node, published to
GitHub Pages at
[mfahad777.github.io/visual-node](https://mfahad777.github.io/visual-node/).

## Local development

```bash
pnpm install
pnpm start
```

Starts a local dev server with live reload.

## Build

```bash
pnpm build
```

Generates static files into `build/`. `onBrokenLinks` is set to `throw`, so a stale
internal link fails this command — always run it locally before pushing.
